# Le_nombre_mystere

Écrire un programme en JAVA pour résoudre ces questions :

a. Ecrire un code qui demande à un utilisateur de fournir un
nombre entier positif compris entre 0 et 100, et ce autant de fois qu'il le souhaite.
On suppose que l'utilisateur fournit correctement les données; 
il n'est donc pas demandé d'écrire un code pour vérifier que les données sont bien rentrées.  

b. Compléter ce code pour qu'il calcule et affiche le nombre d'entiers fournis par l'utilisateur.

c. Compléter ce code pour qu'il calcule et affiche le nombre d'entiers inférieurs ou égaux à 60 fournis par l'utilisateur.
